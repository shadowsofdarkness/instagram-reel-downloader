# Instagram Reel Downloader (simple server)

This project provides a small HTTP endpoint that accepts a public Instagram reel/post URL and returns the direct video URL (if available).

Quick start:
1. Node 18+ recommended
2. npm install
3. node src/index.js
4. GET /api/video?url=https://www.instagram.com/reel/XXXXXXXXX/

Notes:
- Works only for public posts. Scraping may violate Instagram TOS.
